# Documentação WDNA — Site do Ar

Esta pasta guarda a **estrutura real do tema** no painel WDNA.

**Sempre consulte antes de pedir arquivos ao usuário:**

- [`ESTRUTURA_TEMA.md`](ESTRUTURA_TEMA.md) — lista completa de pastas e arquivos que existem no editor WDNA
- [`ARQUIVOS_DO_REPO.md`](ARQUIVOS_DO_REPO.md) — o que este repositório Git altera vs o que fica só na WDNA

## Regras importantes

1. **Não existe pasta `pages/`** no tema WDNA desta loja.
2. **Não inventar arquivos** que não estão em `ESTRUTURA_TEMA.md`.
3. Arquivos como `theme.js`, `theme-defer.js` ficam na WDNA — **não estão no GitHub** (só referência).
4. Correção de preço logado / carrinho: priorizar **Configurações → Produto → Preço** no painel WDNA.
